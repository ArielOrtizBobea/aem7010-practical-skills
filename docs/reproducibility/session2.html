Session 2: Reproducibility

Foundations for reproducible research (with Lars Vilhuber)

Materials for this session are being prepared for the companion site. This page will expand ahead of the session.

Where this session fits

Guest-led by Lars Vilhuber, AEA Data Editor. This first reproducibility session opens the block and sets up vocabulary and expectations.

Topics expected to be covered:

  • Why reproducibility is a research-integrity issue, not a box-check.
  • What top journals now require (AEA, REStud, JPE).
  • openICPSR and the replication-package workflow.
